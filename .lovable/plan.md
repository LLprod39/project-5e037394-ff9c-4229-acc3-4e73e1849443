

# Центр развития особенных детей — Онлайн-тест для родителей

## Концепция
Mobile-first веб-приложение с ощущением нативного приложения. Тёплый, спокойный дизайн, вызывающий доверие. Мягкие пастельные тона (персиковый/лавандовый акценты на светлом фоне), крупные скруглённые карточки, плавные анимации переходов.

## Цветовая палитра
- Фон: тёплый белый (#FDFBF7)
- Основной акцент: мягкий тёплый синий (#6B9ECF)
- Вторичный: лавандовый (#B8A9C9)
- Успех/позитив: мятный (#7EC8A0)
- Предупреждение: тёплый оранжевый (#E8A87C)
- Текст: тёмно-серый (#2D3436)

## Экраны и навигация

### 1. Экран приветствия (`/`)
- Иллюстративная иконка, заголовок, подзаголовок
- Крупная кнопка «Начать тест»
- Мелким шрифтом disclaimer

### 2. Форма данных ребёнка
- Имя ребёнка, возраст, имя родителя, телефон
- Чекбокс согласия на обработку персональных данных
- Sticky-кнопка «Далее»

### 3. Многошаговый тест (3 блока)
- Progress bar сверху (sticky)
- Название текущего блока
- Вопросы по 1-2 на экран как selectable cards
- Single choice и multiple choice
- Sticky-кнопка «Далее» внизу
- Плавные анимации переходов между шагами

**Блок 1: Беременность и роды** — 2 вопроса (расширяемо)
**Блок 2: Физическое и моторное развитие** — заготовка с примерами
**Блок 3: Речь, коммуникация, социальное взаимодействие** — заготовка с примерами

### 4. Экран результата
- Иконка + краткая рекомендация (одна из 4 категорий)
- Бережное объяснение: какие факторы повлияли
- CTA «Записаться на консультацию»
- Кнопка «О центре»

### 5. Страница «О центре» (`/about`)
- Краткое описание центра
- Контактная информация
- Кнопка связи

### 6. Админ-панель (`/admin`)
- Список всех анкет (дата, имя, рекомендация)
- Детальный просмотр анкеты: все ответы, факторы риска, рекомендация с объяснением
- Экспорт отчёта в PDF
- Mock data для демонстрации

## Архитектура данных

### Конфиг вопросов (`questionsConfig.ts`)
```ts
{
  id, blockId, text, type: 'single' | 'multiple',
  options: [{ id, label, riskLevel: 'low' | 'medium' | 'high', riskWeight }]
}
```

### Модуль расчёта результата (`calculateResult.ts`)
- Суммирует risk-weight по всем ответам
- Определяет категорию рекомендации
- Генерирует текстовое объяснение на основе факторов

### Хранение данных
- localStorage + React state (mock)
- Архитектура готова к подключению Supabase

## Компонентная структура
- `WelcomeScreen` — приветствие
- `UserInfoForm` — данные ребёнка/родителя
- `TestFlow` — оркестратор теста
- `QuestionCard` — отображение вопроса
- `SelectableOption` — карточка варианта ответа
- `ProgressBar` — индикатор прогресса
- `ResultScreen` — результат для родителя
- `AboutPage` — о центре
- `AdminDashboard` — список анкет
- `AdminReport` — детальный отчёт
